<h1>🛠️ Installation Steps</h1>
<p>1.Clone the repository</p>

```
[git clone https://github.com/Sandy712/Text-Counter]
```
<p>2.Change the working directory</p>

``` 
[cd Text-Counter]
```
<p>3.Install dependencies</p>

``` 
[npm install]
```
<p>4.Run the app</p>

```
[npm start]

```
